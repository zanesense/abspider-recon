Performance Tips

  • Start with 5 threads (default).
  • Increase up to 20 for speed; use proxy rotation for large scans.
  • Disable unused modules to reduce noise.
  • Data stored safely in browser localStorage only.

ü©∫ Troubleshooting & Best Practices

Scan fails immediately

Check target URL format and ensure accessibility. Reduce module count or thread number. See browser console for errors.

No Subdomains Found

Target may have few subdomains or block enumeration. Try enabling proxy rotation or scan again later.

Discord Webhook Not Working

Verify webhook URL and permissions. Use the ‚ÄúTest Webhook‚Äù button before running scans.

Headers Not Loading

CORS or server rules may block HEAD requests. Enable proxy mode for better results.

Best Practices

  • Start small with test domains.
  • Respect rate limits ‚Äî avoid aggressive scans.
  • Use only with permission.
  • Download and archive PDF reports.
